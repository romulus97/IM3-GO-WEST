eGRID2018 Data was orginally released on 1/28/2020.
eGRID2018v2 Data was released on 3/9/2020.

The following files representing eGRID2018 are available on the eGRID web site (epa.gov/energy/egrid):

			egrid2018_data_v2.xlsx
                     	egrid2018_technical_support_document.pdf
			egrid2018_summary_tables.pdf
			egrid2018_summary_tables.xlsx
			egrid2016_egrid_subregions.jpg
			egrid2016_nerc_regions.jpg		
			egrid2018_data_metric.xlsx
			egrid2018_release_notes_2020-01-28.txt (this file)
			egrid2018_all_files.zip			
			egrid2018_historical_files_since_1996.zip
			egrid2018_sequence_numbers.xlsx


egrid2018_all_files.zip contains files released for eGRID2018, including:

			egrid2018_data_v2.xlsx
			egrid2018_release_notes_2020-01-28.txt (this file)
			egrid2018_summary_tables.pdf
			egrid2018_summary_tables.xlsx
                     	egrid2018_technical_support_document.pdf
						

egrid2018_historical_files_since_1996.zip contains all files from all historical editions of eGRID including data for the following years:
  1996, 1997, 1998, 1999, 2000, 2004, 2005, 2007, 2009, 2010, 2012, 2014, and 2016.
